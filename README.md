# Spoke

A native macOS app that turns text into spoken audio. Drop a file or paste text, pick a speed, preview, and export to M4A.

**[Download](https://galileilabs.github.io/spoke/)**

## Features

- Drag & drop or paste text input
- Speed control (0.75x – 2x)
- Preview with your Mac's system voice
- Export to M4A

## Requirements

- macOS 13 Ventura or later

## Building

Open `Spoke.xcodeproj` in Xcode and build. To create a DMG for distribution:

```
./build-dmg.sh
```
